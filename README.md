# ChatGPT User Productivity Analysis

This project explores a ChatGPT user productivity dataset using Python. The goal is to understand user behavior, usage patterns, and productivity through exploratory data analysis (EDA).

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Project Overview
- Loaded and cleaned the dataset
- Explored user demographics and usage
- Created visualizations to identify trends
- Generated insights from the data

## Files
- `chatgpt_user_productivity_analysis.ipynb` – Jupyter notebook containing the complete analysis.
- `chatgpt_user_productivity_dataset_300.csv` – Dataset used for the project.
